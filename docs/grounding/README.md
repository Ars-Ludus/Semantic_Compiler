# Grounding Directory

This directory contains distilled technical knowledge and patterns for external dependencies and core libraries used in the `semcom` project.

The purpose of grounding is to provide "anchors" of knowledge that the AI agent can use to ensure consistency and idiomatic usage across the codebase.

## Contents

- [Go slog](./go-slog.md): Patterns and conventions for using the Go standard library's structured logging package.
